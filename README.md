## Running The Harness

The simplest way to boot this project is:

1. Install Java.
2. Install KasparBot.zip
3. Extract the zip into a folder.
4. Add KasparBot folder into the harness folder.
5. Launch it from a terminal:

java -jar harness.jar

`harness.jar` automatically picks up bot jars from the folders inside this directory, including `KasparBot/KasparBot.jar`.

## Requirements

- Java
- Harness folder
