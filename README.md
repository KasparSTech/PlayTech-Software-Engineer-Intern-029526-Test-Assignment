# Running The Harness

The simplest way to boot this project is:

1. Install Java.
2. Install KasparBot
3. Add KasparBot into the harness folder.
4. Launch it from a terminal:

java -jar harness.jar

`harness.jar` automatically picks up bot jars from the folders inside this directory, including `KasparBot/KasparBot.jar`.
